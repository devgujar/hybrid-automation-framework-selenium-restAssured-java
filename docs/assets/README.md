# Chatbot assets

Save the robot image you want as the chat icon here, named exactly:

    chatbot.png

Path used by the page: docs/assets/chatbot.png

Both the floating button and the chat header avatar load this file. If the file
is missing, the widget falls back to the 💬 / 🤖 emoji automatically.

Tip: a square image (e.g. 256x256) looks best since it's shown in a circle.
